# Weather Information App (JavaFX)

A JavaFX desktop application that retrieves real-time and forecast weather data using the OpenWeatherMap API.

## Features
- Real-time weather information (temperature, humidity, wind speed)
- Short-term weather forecast
- City search with validation
- Unit conversion (Celsius/Fahrenheit, m/s/mph)
- Search history with timestamps
- Dynamic background based on time of day
- Asynchronous API requests using multithreading

## Technologies Used
- Java (JDK 11+)
- JavaFX
- REST API (OpenWeatherMap)
- HTTP Client
- Object-Oriented Programming
